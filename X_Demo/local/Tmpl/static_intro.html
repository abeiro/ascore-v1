1. Introducción

En el mercado tecnológico actual, es de suma importancia la creación de aplicaciones de manera rápida para eliminar costes, sin perder un ápice de potencia. El uso de tecnologías libres, la reutilización de código y el uso de las herramientas adecuadas acelera el desarrollo de los proyectos.

asCore es un conjunto de clases y librerias diseñadas para facilitar el desarrollo de aplicaciones WEB, si bien puede adaptarse al uso en Intranet Corporativas e incluso aplicaciones personales. Es multiplataforma al basarse en tecnologías multiplataforma. (PHP, MySQL, HTML, JavaScript, Apache)

El objetivo principal de asCore es apoyar al desarrollador liberandolo de tareas tediosas y repetitivas, creando un código limpio separando lenguajes, manteniendo una estructura modular y proporcionando librerías para las tareas más comunes. En la versión actual (1) disponemos de la siguientes funcionalidades:

  1. Abstracción sobre BBDD
  2. Definición dinámica de clases
  3. Operaciones básicas sobre clases en BBDD
  4. Librería de plantillas
  5. Librería de DataGrids
  6. Librería de control de fechas
  7. Librería de Gráficos de Datos
  8. Librería de Selección de Datos Avanzados
  9. Autentificación y control de Usuarios
  10. Panel de Control de Desarrollo
